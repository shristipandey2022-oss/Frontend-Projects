# Rock Paper Scissors Game 

A simple and interactive Rock Paper Scissors game built using **HTML, CSS, and JavaScript**.  
The game allows a user to choose between rock, paper, or scissors, while the computer makes a random choice. Scores are updated dynamically based on the result of each round.



# Features
- User vs Computer gameplay
- Random computer choice generation
- Real-time score updates
- Clear win, lose, and draw messages
- Simple and beginner-friendly logic
- Interactive UI with clickable choices



# Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)



# How it Works
1. Click on **Rock**, **Paper**, or **Scissors**
2. The computer will randomly select its move
3. The result of the round is displayed
4. Scores are updated automatically
5. Same choices result in a draw
