A simple currency converter web application built using **HTML, CSS, and JavaScript**.  
The project allows users to enter an amount, select two different countries using dropdown menus, and convert the amount based on the selected currencies.



# Features
- User input for amount to be converted
- Two dropdown menus to select source and target countries
- Dynamic currency conversion on button click
- Clean and simple user interface
- Beginner-friendly JavaScript logic



# Tech Stack
- HTML
- CSS
- JavaScript 



# How It Works
1. User enters an amount in the input field
2. User selects the **from** and **to** countries using dropdown menus
3. Clicks on the **Convert** button
4. The converted amount is displayed below
