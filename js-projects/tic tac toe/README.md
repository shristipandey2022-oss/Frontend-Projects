# Tic Tac Toe Game 

A classic Tic Tac Toe game built using **HTML, CSS, and JavaScript**.  
The game supports two players playing on the same device, with turn-based logic and automatic winner detection.

---

## Features
- Two-player gameplay (Player X and Player O)
- Automatic winner detection
- Displays winning message
- Reset and New Game functionality
- Clean and playful UI design
- Beginner-friendly JavaScript logic

---

## Technologies Used
- HTML
- CSS
- JavaScript (Vanilla JS)

---

## How to Play
1. Player O starts the game
2. Players take turns clicking on the grid
3. The first player to align three symbols wins
4. Use **Reset** or **New Game** to restart

---

## Winning Conditions
A player wins if they get three symbols in:
- A row
- A column
- A diagonal
